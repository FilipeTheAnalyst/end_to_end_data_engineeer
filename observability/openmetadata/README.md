# Observability (OpenMetadata)

OpenMetadata is used for catalog + lineage across:
- Snowflake
- dbt
- Metabase
- (optional) Airbyte

This repo includes ingestion YAML templates in `ingestion/`.

## Running OpenMetadata
OpenMetadata is usually started using its official Docker Compose.
This repo keeps only configuration templates; integrate the official compose here if desired.

Docs: https://docs.open-metadata.org/
